# Stripe Product Description — RiverScout Pro

The Stripe product description lives in Stripe's dashboard, not in the repo.
After updating the Pro positioning in the app, copy the text below into the
RiverScout Pro product in the Stripe dashboard so the Checkout page, invoices,
and customer portal all reflect the new model.

## Where to update

1. Stripe Dashboard → **Products** → **RiverScout Pro** → Edit
2. Replace the **Description** field with the block under "Product description" below
3. Save

Both the monthly and yearly prices use the same product, so you only need to
edit the description once.

---

## Product description (paste into Stripe)

```
RiverScout Pro — The site works for you

Get notified automatically when your rivers hit optimal conditions, when
they're stocked, and when the hatch is about to start. Plus offline access,
historical flow data, and AI-powered forecasts.

Everything in the free tier plus:
- Flow alert emails (unlimited)
- Stocking alert emails
- Hatch alert emails
- Offline river pages
- 10-year historical flow analysis
- AI forecast interpretation
- Custom CFS ranges
- River journal and trip statistics
```

---

## Context

As of April 9, 2026 RiverScout shifted to a generous free tier:

- All conditions data, forecasts, stocking and hatch data, weather, trip
  reports, and unlimited flow alert **subscriptions** are now fully free.
- Pro is only for users who want the site to actively come to them —
  email delivery, offline access, and deeper analytics.

Keeping the Stripe description in sync with the in-app positioning avoids
a confusing checkout experience where users see different promises in the
app vs the Stripe-hosted page.
